# 个人博客项目修改文档
## 一、项目概述
基于原有的 Next.js + Prisma 博客模板，扩展了以下核心功能：
1. 个人资料页面 - 支持查看和编辑用户资料
2. 分类页面 - 展示所有文章分类
3. 标签页面 - 展示标签云

## 三、功能详解
### 3.1 个人资料页面
功能特性：
- 用户资料展示（头像、昵称、标签、签名）
- 关于我模块（支持富文本编辑）
- 基本信息展示（所在地、职业、开通时间）
- 联系方式（GitHub、微信、个人网站）
- 博客统计（已发布文章数、草稿数）
- 最新文章列表
编辑权限：
- 只有登录用户可以编辑自己的资料
- 其他用户只能查看
API 端点：
- GET /api/users/[id] - 获取用户资料
- PATCH /api/users/[id] - 更新用户资料（需登录且为本人）
### 3.2 分类页面
功能特性：
- 展示所有文章分类
- 显示每个分类的文章数量
- 点击分类可查看该分类下的文章
### 3.3 标签页面
功能特性：
- 展示所有标签云
- 点击标签可查看该标签下的文章
### 3.4 数据初始化
使用方式：
```
# 访问以下 URL 初始化数据
GET /api/seed
```
初始化内容：
- 常见文章分类（技术、生活、随笔、教程等）
- 常见标签

## 五、运行方式
```
# 安装依赖
npm install

# 生成 Prisma 客户端
npx prisma generate

# 同步数据库 Schema
npx prisma db push

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 七、技术栈
- 框架 : Next.js 16.2.5 (Turbopack)
- 数据库 : PostgreSQL + Prisma ORM
- 认证 : NextAuth.js
- 样式 : Tailwind CSS 3
- 语言 : TypeScript
## 八、注意事项
1. 首次运行需配置 .env 文件，设置数据库连接字符串
2. 运行 npx prisma db push 同步数据库 Schema
3. 访问 /api/seed 初始化分类和标签数据
4. 用户注册后可编辑自己的个人资料
