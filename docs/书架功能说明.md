# 书架功能说明

本文档说明「书架」页如何收藏喜欢的书、管理阅读状态与封面展示，便于使用与维护。

## 功能概览

- **入口**：导航栏「书架」（位于「文章」与「音乐」之间）→ `/bookshelf`。
- **权限**：**仅登录用户**可访问与管理；藏书列表为**用户私有**。
- **展示**：暖色书架背景 + 分层搁板；每行最多 4 本书的卡片网格。
- **阅读状态**：想读 / 在读 / 已读，支持筛选与卡片内快速切换。
- **封面**：可填图片直链；无封面时按书名生成渐变色块；外链图经同源代理加载。
- **其它**：可选 1～5 星评分、短评、外链（豆瓣购书页等）。

## 添加书籍

在页面顶部「放上新书」区域填写：

| 字段 | 必填 | 说明 |
|------|------|------|
| 书名 | 是 | 卡片主标题 |
| 作者 | 否 | 副标题显示 |
| 封面图片链接 | 否 | **图片直链**（见下文「封面不显示」） |
| 短评 / 读书笔记 | 否 | 卡片内最多显示两行 |
| 阅读状态 | 否 | 默认「想读」 |
| 评分 | 否 | 1～5 星，可不选 |
| 外链 | 否 | 豆瓣详情、购书页等；卡片显示「查看详情」 |

点击「放进书架」→ `POST /api/bookshelf`。

## 筛选与管理

- 顶部分类：**全部**、**在读**、**想读**、**已读**（带数量），URL 形如 `/bookshelf?status=READING`。
- 卡片上可改「阅读状态」下拉框，即时 `PATCH /api/bookshelf/[id]`。
- 「移除」从书架删除，需确认。

## 封面不显示时怎么办

常见原因是粘贴了**书籍网页地址**而非**封面图片地址**，或图床防盗链。

1. 在豆瓣等站点打开书籍页，**右键封面图 → 复制图片地址**。
2. 粘贴到「封面图片链接」，格式类似：  
   `https://img9.doubanio.com/view/subject/s/public/sxxxx.jpg`
3. 不要粘贴：`https://book.douban.com/subject/123456/`（这是网页）。

站内展示时，外链封面会经 **`GET /api/bookshelf/coverProxy?src=…`** 由服务器拉取（豆瓣等会带合适 Referer），避免浏览器直连被拦。

保存时 `normalizeCoverUrl` 会自动补全 `https://` 并校验协议。

## 数据模型（Prisma）

`BookshelfItem` 字段要点：

| 字段 | 说明 |
|------|------|
| `title` | 书名 |
| `author` | 作者 |
| `coverUrl` | 封面原图 URL（库内存原始地址） |
| `note` | 短评 |
| `status` | `WANT_TO_READ` \| `READING` \| `FINISHED` |
| `rating` | 1～5，可空 |
| `linkUrl` | 外链 |
| `sortOrder` | 排序（预留） |
| `userId` | 所属用户 |

迁移：`prisma/migrations/20260516120000_add_bookshelf/`。本地也可用 `npx prisma db push` 同步。

## API 摘要

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/bookshelf` | 当前用户藏书列表；`?status=` 可筛状态 |
| POST | `/api/bookshelf` | 添加（body 见 `createBookshelfItem`） |
| PATCH | `/api/bookshelf/[id]` | 更新字段或状态 |
| DELETE | `/api/bookshelf/[id]` | 移除 |
| GET | `/api/bookshelf/coverProxy?src=` | 封面图代理（仅 http/https 外链） |

以上写操作均需登录（`requireAuth`）。

### POST 请求体示例

```json
{
  "title": "三体",
  "author": "刘慈欣",
  "coverUrl": "https://img9.doubanio.com/view/subject/s/public/sxxxx.jpg",
  "note": "科幻入门必读",
  "status": "READING",
  "rating": 5,
  "linkUrl": "https://book.douban.com/subject/2567698/"
}
```

## 路由与登录保护

- 页面：`src/app/bookshelf/page.tsx`（`BookshelfContent` + `Suspense` 包裹筛选）。
- 未登录访问 `/bookshelf` 会重定向到 `/login?callbackUrl=/bookshelf`。
- `src/proxy.ts` 的 `matcher` 包含 `/bookshelf`、`/bookshelf/:path*`。

## 维护注意

- 修改 `schema.prisma` 后需执行 `npx prisma generate`；若 dev 报 `bookshelfItem` 为 `undefined`，先停掉 `npm run dev` 再生成。
- API 路由文件夹名使用 **`coverProxy`**（驼峰），避免 Next.js 16 下带连字符的路由名引发类型生成异常（与音乐 `audioProxy` 相同约定）。
