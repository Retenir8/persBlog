# 文章功能说明

本文档说明博客文章的发布、浏览、分类标签、评论、点赞与草稿等能力。

## 功能概览

| 能力 | 说明 |
|------|------|
| 全站动态流 | 首页 `/`、文章列表 `/posts` 展示**已发布**文章 |
| 我的文章 | `/myposts` 登录后管理自己的已发布与草稿 |
| 发布 / 编辑 | `/posts/new`、`/posts/[id]/edit` |
| 详情 | `/posts/[id]`：正文、分类标签、浏览量、点赞、评论 |
| 分类与标签 | 每用户私有，在 `/categories`、`/tags` 管理 |
| 搜索 | 列表支持关键词筛选（标题、正文） |
| 猫咪读文 | 详情页「让猫咪读一读」（需登录，见 `docs/宠物AI功能说明.md`） |

## 权限与可见性

### 已发布文章（`published: true`）

- 所有人可阅读详情与评论（评论是否允许游客见 `CommentSection` 的 `allowGuest`）。
- 出现在首页与 `/posts` 时间线。

### 草稿（`published: false`）

- 仅**作者本人**与 **ADMIN** 可查看详情；他人访问为 404。
- 详情页顶部显示「草稿 · 仅作者与管理员可见」提示条。

### 发布与编辑

- 新建、编辑需登录；编辑页仅作者或 ADMIN 可进入。
- ADMIN 编辑他人文章时，分类/标签校验以**文章作者**的私有分类标签为准。

## 写文章

编辑器使用 **TipTap**（`src/components/blog/Editor.tsx`），保存 HTML 到 `Post.content`。

发布表单（`PostEditorForm`）包含：

- 标题
- 富文本正文
- 分类（单选，可先快捷新建）
- 标签（多选，可先快捷新建）
- 发布状态：发布 / 保存为草稿

提交：`POST /api/posts`（新建）或 `PATCH /api/posts/[id]`（编辑）。

## 文章详情页

- 元信息：作者、发布时间、浏览量、点赞、分类与标签。
- 顶栏：返回列表、「让猫咪读一读」（登录）、编辑（有权限时）。
- 正文：`post-content` 样式渲染 HTML。
- **评论**：`CommentSection`，支持多级回复；登录用户可评，已发布文章可按配置允许游客。
- **浏览量**：进入详情时客户端上报 `POST /api/posts/[id]/view` 累加。
- **点赞**：登录用户可切换点赞，`POST` / 查询状态见 like 路由。

## 分类与标签

- **私有**：`Category`、`Tag` 均绑定 `userId`，不可使用他人的分类/标签。
- 管理页：`/categories`、`/tags`（需登录，`proxy.ts` 中配置）。
- 文章保存时服务端校验分类/标签归属。

## 评论（文章下）

与留言板不同，评论挂在 `Post` 上：

- 树形结构：`parentId` 指向父评论，支持回复。
- 删除、权限逻辑见 `commentService` 与 `CommentSection`。
- API：`/api/posts/[id]/comments` 等。

## 管理端

- 路径：`/admin`（仅 `role === ADMIN`）。
- 可管理全站文章列表（见 `src/app/admin/page.tsx`）。

## 数据模型（Prisma）摘要

| 模型 | 说明 |
|------|------|
| `Post` | 标题、HTML 正文、`published`、`authorId`、`categoryId`、`viewCount` |
| `Category` / `Tag` | 用户私有分类与标签 |
| `PostTag` | 文章与标签多对多 |
| `Comment` | 文章评论（可嵌套） |
| `Like` | 用户对文章点赞，`(postId, userId)` 唯一 |

## API 摘要（常用）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/posts/search` | 公开列表搜索 |
| GET | `/api/posts/mine` | 当前用户文章（含草稿） |
| POST | `/api/posts` | 创建文章 |
| GET/PATCH/DELETE | `/api/posts/[id]` | 读写单篇 |
| POST | `/api/posts/[id]/view` | 增加浏览量 |
| POST | `/api/posts/[id]/like` | 点赞/取消 |
| GET/POST | `/api/posts/[id]/comments` | 评论列表与发表 |
| GET/POST | `/api/categories`、`/api/tags` | 分类标签管理 |
