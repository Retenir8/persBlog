# 个人博客项目修改文档
## 一、项目概述
基于原有的 Next.js + Prisma 博客模板，扩展了以下核心功能：
1. 个人资料页面 - 支持查看和编辑用户资料
2. 分类页面 - 展示所有文章分类
3. 标签页面 - 展示标签云
4. 文章阅读量统计 - 防重复计数
5. 点赞功能 - 支持点赞/取消点赞
6. 评论系统 - 支持访客评论和回复
7. 文章搜索功能 - 按标题和内容搜索

## 三、功能详解
### 3.1 个人资料页面
功能特性：
- 用户资料展示（头像、昵称、标签、签名）
- 关于我模块（支持富文本编辑）
- 基本信息展示（所在地、职业、开通时间）
- 联系方式（GitHub、微信、个人网站）
- 博客统计（已发布文章数、草稿数）
- 最新文章列表
编辑权限：
- 只有登录用户可以编辑自己的资料
- 其他用户只能查看
API 端点：
- GET /api/users/[id] - 获取用户资料
- PATCH /api/users/[id] - 更新用户资料（需登录且为本人）

### 3.2 分类页面
功能特性：
- 展示所有文章分类
- 显示每个分类的文章数量
- 点击分类可查看该分类下的文章

### 3.3 标签页面
功能特性：
- 展示所有标签云
- 点击标签可查看该标签下的文章

### 3.4 数据初始化
使用方式：
```
# 访问以下 URL 初始化数据
GET /api/seed
```
初始化内容：
- 常见文章分类（技术、生活、随笔、教程等）
- 常见标签

### 3.5 文章阅读量统计
功能特性：
- 进入文章详情页自动增加阅读量
- 同一用户对同一文章只计一次（基于 sessionStorage）
- 列表页和详情页显示同步（基于 localStorage 缓存）

### 3.6 点赞功能
功能特性：
- 支持文章点赞/取消点赞切换
- 实时显示点赞数量
- 基于用户 ID 记录点赞状态

### 3.7 评论系统
功能特性：
- 登录用户可直接评论（显示用户名）
- 访客评论需填写昵称
- 支持二级评论回复
- 作者和管理员可删除评论

### 3.8 文章搜索
功能特性：
- 支持按标题搜索
- 支持按内容搜索
- 大小写不敏感匹配

## 四、数据库 Schema 变更
### 4.1 User 模型扩展
```
model User {
  // 原有字段...
  bio        String?  // 个人简介
  signature  String?  // 个人签名
  tags       String?  // 个人标签（逗号分隔）
  location   String?  // 所在地
  occupation String?  // 职业
  github     String?  // GitHub 链接
  wechat     String?  // 微信号
  website    String?  // 个人网站
}
```

### 4.2 Post 模型扩展
```
model Post {
  // 原有字段...
  viewCount Int @default(0)  // 浏览量
}
```

### 4.3 Like 模型（新增）
```
model Like {
  id     String @id @default(cuid())
  userId String
  postId String
  user   User @relation(fields: [userId], references: [id])
  post   Post @relation(fields: [postId], references: [id], onDelete: Cascade)
  
  @@unique([userId, postId])
}
```

### 4.4 Comment 模型（新增）
```
model Comment {
  id        String    @id @default(cuid())
  content   String
  postId    String
  post      Post      @relation(fields: [postId], references: [id], onDelete: Cascade)
  userId    String?
  user      User?     @relation(fields: [userId], references: [id])
  guestName String?
  parentId  String?
  parent    Comment?  @relation("CommentReplies", fields: [parentId], references: [id])
  replies   Comment[] @relation("CommentReplies")
  createdAt DateTime  @default(now())
}
```

## 五、运行方式
```
# 安装依赖
npm install

# 生成 Prisma 客户端
npx prisma generate

# 同步数据库 Schema
npx prisma db push

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 七、技术栈
- 框架 : Next.js 16.2.6 (Turbopack)
- 数据库 : PostgreSQL + Prisma ORM
- 认证 : NextAuth.js
- 样式 : Tailwind CSS 4
- 语言 : TypeScript

## 八、注意事项
1. 首次运行需配置 .env 文件，设置数据库连接字符串
2. 运行 npx prisma db push 同步数据库 Schema
3. 访问 /api/seed 初始化分类和标签数据
4. 用户注册后可编辑自己的个人资料
5. 评论功能支持登录用户和访客两种模式
6. 阅读量统计基于 sessionStorage，同一会话不重复计数
